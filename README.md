# Docker-Ollama

Docker-Ollama is a lightweight local setup for running Ollama inside Docker using Docker Compose.  
This project provides a simple, reproducible environment to start and manage an Ollama container for local LLM development, testing, and experimentation without installing Ollama directly on the host machine.

It is intended for developers who want:
- A quick local Ollama environment with minimal setup
- Consistent behavior across machines via containerization
- Easy start/stop workflow using Docker Compose
